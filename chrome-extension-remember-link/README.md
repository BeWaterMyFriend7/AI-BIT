# 链接收藏管理器

专为内网环境设计的 Chrome 浏览器插件，实现一键保存和导入浏览器标签页链接。

## 功能特点

- **一键保存**: 点击按钮自动保存所有浏览器窗口的标签页链接
- **智能去重**: 自动去除重复的链接
- **双重存储**: 支持保存到文本文件或浏览器书签文件夹
- **一键导入**: 从文件或书签文件夹快速恢复所有链接
- **批量打开**: 导入后自动在新标签页中打开所有链接
- **内网友好**: 无需外网访问，完全离线工作

## 使用方法

### 安装步骤

1. 下载或复制本项目到本地文件夹
2. 打开 Chrome 浏览器
3. 在地址栏输入: `chrome://extensions/`
4. 开启右上角的 **"开发者模式"**
5. 点击 **"加载已解压的扩展程序"**
6. 选择本项目文件夹即可

### 保存链接

1. 点击浏览器工具栏中的插件图标
2. 查看当前标签页数量
3. 点击 **"保存链接"** 按钮
4. 在预览区域确认链接列表
5. 选择保存方式:
   - **保存到文件**: 下载为 .txt 文本文件
   - **保存到书签**: 保存到指定的书签文件夹
6. 点击 **"确认保存"**

### 导入链接

**方式一：从文本文件导入**
1. 点击 **"导入链接"** 按钮
2. 选择 **"从文本文件"**
3. 点击 **"选择文件"** 按钮
4. 选择包含链接的 .txt 文件
5. 插件会自动打开文件中的所有链接

**方式二：从书签导入**
1. 点击 **"导入链接"** 按钮
2. 选择 **"从书签文件夹"**
3. 输入之前保存的书签文件夹名称（默认: tmp）
4. 点击 **"选择文件"** 按钮（触发导入）
5. 插件会自动打开书签中的所有链接

## 技术规格

- **Manifest 版本**: 3
- **权限需求**: tabs, downloads, bookmarks
- **兼容性**: Chrome 88 及以上版本
- **文件编码**: UTF-8

## 注意事项

1. 插件会自动过滤非 HTTP/HTTPS 链接（如 chrome://、file:// 等）
2. 保存到书签时会自动清空同名文件夹后重新添加
3. 导入时会按顺序在新标签页中打开所有链接
4. 建议定期清理不需要的标签页以减少噪音

## 常见问题

**Q: 保存文件时提示错误？**
A: 确保 Chrome 的下载功能正常，尝试允许下载多个文件。

**Q: 无法找到保存的书签？**
A: 检查书签管理器（Ctrl+Shift+O）中是否存在指定名称的文件夹。

**Q: 导入的文件格式？**
A: 文件应为纯文本，每行一个完整 URL（以 http:// 或 https:// 开头）。
