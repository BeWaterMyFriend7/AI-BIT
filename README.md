mini PROJECT by AI。一些AI写的小项目，实现一些想法，解决一些需求。

## 项目列表

  记录访问链接的快照的浏览器插件，详细说明 [link](chrome-extension-remember-link/README.md)

- skill-bridge
  本地 Agent Skill 共享管理器。管理多个 AI 编码 Agent (Codex, OpenCode 等) 的 Skill 目录，
  通过 NTFS Junction 实现"一份 Skill，多 Agent 使用"，避免重复安装和版本漂移。
